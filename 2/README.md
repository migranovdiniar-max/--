# Telegram Text Sender

Простой скрипт на Python, который читает текст из файла `.txt` и отправляет его в выбранный приватный чат Telegram через вашего бота.

## Что делает скрипт

- Читает содержимое указанного файла (UTF-8)
- Отправляет весь текст одним сообщением в Telegram-чат
- Поддерживает .env для безопасного хранения токена и chat_id
- Обрабатывает базовые ошибки (файл не найден, пустой текст, проблемы с API)

## Требования и зависимости

- Python 3.8+
- Библиотеки:
  ```bash
  pip install python-telegram-bot python-dotenv

# Настройка
Создайте Telegram-бота:
- Напишите @BotFather
- /newbot, задайте имя, получите токен 
- Получите CHAT_ID (ID приватного чата):
- Напишите своему боту любое сообщение (например /start)
- Откройте в браузере: https://api.telegram.org/botВАШ_ТОКЕН/getUpdates
- Найдите в JSON "chat":{"id":123456789,... 
Создайте файл .env в той же папке со скриптом:
